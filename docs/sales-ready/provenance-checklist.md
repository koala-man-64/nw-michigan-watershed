# Dataset and Customer Pack Provenance Checklist

## Dataset Pack
- Confirm dataset owner and source organization
- Confirm allowed use and transfer rights
- Record schema version and checksum
- Record source files included in the pack
- Record validation status and publish eligibility

## Customer Pack
- Confirm branding ownership or license
- Confirm support contact approval
- Confirm legal/privacy links
- Confirm feature flags enabled for the customer
- Confirm deployment environment and release mapping

## Pre-Sale Gate
- Platform core separated from customer-specific assets
- Dataset rights documented
- Environment secrets excluded from repository transfer
- Third-party dependency notices prepared
- Release artifact inventory captured
